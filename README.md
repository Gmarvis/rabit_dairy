# Rabbit Dairy

Personal financial tracking app — React Native (Expo) + Supabase.
Log spending three ways: type it, speak it (with a voice reason), or scan a
bank / mobile-money statement.

## Design

- `design/ui-flow.html` — full UI flow and screen-by-screen mockups (open in a browser).

Currency: XAF (FCFA). Built as an npm-workspaces monorepo using DDD + CQRS.
